Política de Privacidad

botgod.pro (God Bot) respeta tu privacidad. Esta política describe qué datos recopilamos y cómo los usamos.

Datos que Recopilamos

  • IDs de Discord: IDs de usuarios, servidores, canales y roles necesarios para el funcionamiento del bot.
  • Configuración del servidor: preferencias, warns, tickets, XP, economía y otros datos de módulos.
  • Sesión web: si inicias sesión en el dashboard, almacenamos una sesión temporal vía OAuth2 de Discord.
  • Transcripts: si usas el módulo de tickets, se guardan registros de conversaciones.

Cómo Usamos los Datos

  • Para proporcionar las funcionalidades del bot (moderación, niveles, economía, música, etc.).
  • Para el dashboard web que permite configurar el bot.
  • No vendemos ni compartimos tus datos con terceros.

Almacenamiento

  • Los datos se almacenan en una base de datos SQLite local del operador.
  • No utilizamos servicios de análisis de terceros.
  • Los datos se mantienen mientras el bot esté activo.

Tus Derechos

  • Puedes solicitar la eliminación de tus datos contactando al operador.
  • Puedes eliminar al bot de tu servidor en cualquier momento.

Contacto

Para preguntas sobre privacidad, contacta al operador del bot.
